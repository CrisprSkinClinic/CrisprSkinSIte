Backup Encryption Keys

This exports the master keys that decrypt every patient's personal information (name, phone, DOB, address, etc.) to Google Drive, protected with a password you choose here.

Important: if you forget this password, the backup file cannot be recovered by anyone -- not even Anthropic, Claude, or Supabase support. Store it somewhere safe (a password manager), separate from this backup file itself.

You must be signed in as a doctor on this device (via /prescription) for this to work.
